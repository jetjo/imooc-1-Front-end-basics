快捷键

                按住Shift输入大写字母，单击Shift切换中英文
                
                按住Ctrl可以在文档中多处插入光标
                
                同时按住Ctrl和Shift，点击文档某处，可以在其实光标和点击行之间所有行中插入光标
                
                Ctrl + F2：首先选中一段文本，同时按住以进入批量修改模式
                
                Ctrl + ~：打开内置终端
                
                Ctrl + Shift + p：打开命令面板
                
                Ctrl + Shift + b：打开系统的特殊字符输入面板

                注：一下快捷键需要安装Sublime Text Keymap and Settings Importer vscode扩展
                
                Ctrl + Shift + d: 复制光标所在行并粘贴到所在行的下一行
                
                Ctrl + Shift + ↑：将光标所在行上移一行
                
                Ctrl + Shift + ↓：将光标所在行下移一行

                注：以下快捷键只会在各自相应的文档类型中有效，实际属于vscode的智能响应功能
                
                !：在空白html文档中输入用以快速生成文档
                
                Vue VSCode Snippets vscode扩展的快捷键
                
                Vue 3 Snippets vscode扩展的快捷键

HTML骨架

                文档第一行
                <!DOCTYPE html>是文档类型定义，DTD
                
                <html lang="en">：html文档自身内容，lang属性指定文档的语言
            
                <head>标签用于放置网页的配置信息，不可见；包括文档的编码、关键词和标题等
            
                <body>文档可见部分

文档编码

                文档的编码信息放在
                
                <head>标签中，使用<meta>标签的[charset]属性指定
                文档的编码类型需要在vscode中指定，在窗口下方的状态栏中
            
                主要使用两种编码utf-8和gbk(gb2312的扩展，以支持繁体字、古文字等不常用字)
                如果网页是中文内容，使用gbk编码可以减小网页的体积，因为utf-8使用3个字节编码汉字，而gbk使用2个字节编码汉字

网页关键词

                使用meta标签在head标签内定义网页关键词，用于网页检索,提升爬虫的抓取质量,如下所示：
                <meta Keywods="关键词1，关键词2">
                <meta Descriptions="描述...">

部分标签

                h系列标签共有六级h1 ~ h6，h1标签每个网页最好只有一个，多于一个时，会被视为作弊。
                与p标签不能相互嵌套
                
                div标签，英文单词分割的简写
