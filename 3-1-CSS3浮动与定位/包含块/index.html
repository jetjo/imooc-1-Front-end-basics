• 1、如果元素的position属性是static、relative、sticky，那么其包含块就是符合如下条件之一的离他最近的祖先元素的content-edge：
    • 1、块元素
    • 2、行内块元素
    • 3、list-item
    • 4、勘误：不是形成BFC的元素，MDN上的原话是“establishes a formatting context (such as a table container, flex container, grid container, or the block container itself)”
  • 2、如果元素的position属性是absolute，那么其包含块就是离他最近的定位属性不是static的祖先元素的padding-edge
  • 3、如果元素的position属性是fixed，或者元素是html，那么其包含块就是viewport（连续的媒体设备）或者page (分页的媒体设备)的padding-edge，即初始包含块
  • 4、如果元素的position属性是absolute或者fixed，那么符合如下条件的距离最近的祖先元素的padding-edge将成为包含块：
    • 1、transform或perspective不是none
    • 2、will-change的值是transform或perspective
    • 3、filter的值不是none或者will-change的值是filter
    • 4、contain的值是paint
    • 5、backdrop-filter的值不是none

可见对于元素自身而言，判断其包含块完全取决于其position属性
